EndlessCreation 2014
=========================


## 사용법
 - 사람 이미지 넣는 곳 : `/img/member/`
 - 사람 추가 방법은 `_members` 폴더를 참고
 - 프로젝트 이미지 넣는 곳 :  `/img/portoflio/`
 - 프로젝트 추가 방법은 `_posts` 폴더를 참고


## 로컬에서 돌리는 방법
Ruby 설치 후

```ruby
gem install jekyll
// jekyll 설치후
git clone https://github.com/ec2014e/ec2014e.github.com.git
cd ec2014e.github.com
jekyll serve
```
